Aplicación Web de Lista de Tareas

Este proyecto consiste en desarrollar una aplicación web simple para gestionar una lista de tareas. Los usuarios podrán añadir, eliminar y marcar tareas como completadas. El enfoque estará en crear una interfaz de usuario clara y amigable, utilizando HTML, CSS y JavaScript básico. Este proyecto ayudará a entender los principios del diseño de interacción y cómo implementar funcionalidades básicas en una aplicación web.

Aplicación Móvil de Reseñas de Restaurantes

Este proyecto implica el desarrollo de una aplicación móvil que permite a los usuarios buscar restaurantes cercanos y escribir reseñas sobre ellos. El proyecto incorporará el uso de una API para obtener datos de restaurantes, como Google Places API. El diseño UX/UI jugará un papel crucial en la creación de una experiencia de usuario fluida y atractiva, centrándose en la facilidad de uso y la accesibilidad.
